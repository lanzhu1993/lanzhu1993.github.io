获取消息列表

接口说明
void getCenterMessages(Integer msgType, int limit, int offset, Condition condition)
参数说明
参数 说明
msgType 消息类别(1:CAMERA,2:SECURITY,3:GENERAL)
limit 限制条数
offset 偏移量
condition 查询筛选条件
示例代码
 TuyaSecurityMessageSDK.newInstance(homeId).getCenterMessages(msgType, limit, offset, condition, new ITuyaResultCallback<ArrayList<CenterMessageBean>>() {
            @Override
            public void onSuccess(ArrayList<CenterMessageBean> result) {
                // do something
            }

            @Override
            public void onError(String errorCode, String errorMessage) {
                // do something
            }
        });
