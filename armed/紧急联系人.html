紧急联系人

安防紧急联系人SDK

ContactBean 字段信息
字段 类型 描述
areaCode String 地区编码
phone String 手机号码
homeId String 家庭ID
firstName String 姓
lastName String 名
id Integer 联系人ID
gmtCreate String 创建时间
sequence Integer 联系人排序

接口说明

            // do something
        }
    });
