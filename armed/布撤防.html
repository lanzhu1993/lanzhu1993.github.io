布撤防管理

安防布撤防SDK依赖

    implementation 'com.tuya.smart:tuyasecurity-basesdk:1.0.0-SNAPSHOT'
GatewayModeBean 字段信息
字段 类型 描述
state int 布撤防执行状态（0:初始状态；1:开始任务；2:继续任务；3:终止任务；4:全部成功；5:全部失败；6:部分异常状态）
mode int 布撤防状态 （0:撤防；1:在家；2:离家）
majorId String 家庭主网关ID
detail List 网关列表详情
GatewayDetailBean 字段信息
字段 类型 描述
gwId String 网关ID
state int 网关状态（0:异常；1:正常）
BypassDetailBean 字段信息
字段 类型 描述
gwId String 网关ID
state int 网关状态（0:异常；1:正常）
bypass List bypass设备nodeID列表
BypassDetailBean 字段信息
字段 类型 描述
gwId String 网关ID
state int 网关状态（0:异常；1:正常）
ignore List 忽略设备设备nodeID列表
HomeInfoBean 字段信息
字段 类型 描述
homeId String 家庭ID
mode int 家庭布撤防状态
alarmState int 家庭报警状态（0:正常；1:报警）
alarmVoice int 家庭报警声音状态（0:静音；1:报警）
alarmCountdown long 报警倒计时
TYSecurityModeErrorType 字段信息
字段 类型 描述
QueryModeError Enum 布防状态查询失败
SwitchModeError Enum 布防状态切换失败
QueryIgnoreDeviceError Enum 忽略设备查询失败
QueryBypassDeviceError Enum Bypass设备查询失败
QueryAbnormalDeviceError Enum 异常设备查询失败
Mqtt 52协议回调监听

注册Mqtt52协议监听器

TuyaSecurityBaseSDK.newInstance(homeId).registerProtocolListener(mTuyaProtocolListener)

销毁Mqtt52协议监听器

TuyaSecurityBaseSDK.newInstance(homeId).unRegisterProtocolListener(mTuyaProtocolListener)

参考接口ITuyaProtocolListener

方法名称 方法描述
onProcessBypassNotice 处理Bypass设备通知
onProcessArmResultNotice 处理布撤防成功通知
onProcessArmCountDownNotice 处理布撤防倒计时通知
onProcessAlarmCountDownNotice 处理报警倒计时通知
onProcessTamperAlarmNotice 处理防拆报警通知
onProcessAddGatewayNotice 处理添加网关通知
onProcessNewAlarmNotice 处理新报警通知
onProcessUpdateAlarmNotice 处理报警更新通知
onProcessAlarmStateNotice 处理报警状态通知
onProcessAlarmVoiceNotice 处理报警声音通知 
public interface ITuyaProtocolListener {

    /**
     * 52通知处理Bypass回调
     *
     * @param result 回调参数
     */
    void onProcessBypassNotice(NormalResult result);

    /**
     * 52通知处理布撤防结果回调
     *
     * @param result 回调参数
     */
    void onProcessArmResultNotice(NormalResult result);

    /**
     * 52通知处理布防倒计时回调
     *
     * @param result 回调参数
     */
    void onProcessArmCountDownNotice(CountDownResult result);

    /**
     * 52通知处理报警倒计时回调
     *
     * @param result 回调参数
     */
    void onProcessAlarmCountDownNotice(CountDownResult result);

    /**
     * 52通知处理防拆报警回调
     */
    void onProcessTamperAlarmNotice();

    /**
     * 52通知处理添加网关回调
     */
    void onProcessAddGatewayNotice();

    /**
     * 52通知处理新报警回调
     *
     * @param result 回调参数
     */
    void onProcessNewAlarmNotice(AlarmResult result);

    /**
     * 52通知处理更新报警消息回调
     */
    void onProcessUpdateAlarmNotice();

    /**
     * 52通知处理报警通知回调
     *
     * @param result 回调参数
     */
    void onProcessAlarmStateNotice(AlarmResult result);

    /**
     * 52通知处理报警声音通知回调
     *
     * @param result 回调参数
     */
    void onProcessAlarmVoiceNotice(AlarmResult result);
}
安防网关监听回调

注册安防网关监听器

TuyaSecurityBaseSDK.newInstance(homeId).registerGatewayListener(mTuyaGatewayListener)

销毁安防网关监听器

TuyaSecurityBaseSDK.newInstance(homeId).unRegisterGatewayListener(mTuyaGatewayListener)

参考接口ITuyaGatewayListener

方法名称 方法描述
onSecurityModeWillSwitchAfterDelay 从网关获取布撤防切换后倒计时
didReceiveUnusualDevice 从云端获取忽略设备列表
onIgnoreDeviceFromGateway 从网关获取忽略设备列表 
public interface ITuyaGatewayListener {

    /**
     * 布撤防切换后倒计时
     *
     * @param countDownTime 倒计时
     */
    void onSecurityModeWillSwitchAfterDelay(int countDownTime);


    /**
     * 获取忽略设备列表
     *
     * @param deviceIds 设备ID集合
     */
    void didReceiveUnusualDevice(ArrayList<String> deviceIds);

    /**
     * 从网关查询忽略设备
     *
     * @param devices 忽略设备JSON数组
     */
    void onIgnoreDeviceFromGateway(JSONArray devices);

}

切换布撤防模式

接口说明
void switchSecurityArmModeWithHomeId(Integer mode, ITuyaResultCallback<StartTidBean> callback)
参数说明
参数 说明
mode 布撤防状态（0:撤防；1:在家；2:离家）
示例代码
 TuyaSecurityBaseSDK.newInstance(homeId).switchSecurityArmModeWithHomeId(mode, new ITuyaResultCallback<StartTidBean>() {
            @Override
            public void onSuccess(StartTidBean result) {
                // do something
            }

            @Override
            public void onError(String errorCode, String errorMessage) {
                // do something
            }
        });
触发接口回调

当切换布撤防过程中，存在忽略设备，则会触发ITuyaGatewayListener.didReceiveUnusualDevice()方法接口回调，并返回忽略设备列表集合

mTuyaGatewayListener.didReceiveUnusualDevice(deviceIds);

否则直接进行布撤防，布撤防成功后会触发ITuyaProtocolListener.onProcessArmResultNotice()方法接口回调

mTuyaProtocolListener.onProcessArmResultNotice(result);
