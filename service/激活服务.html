激活服务

安防增值服务SDK

DealerBean 字段信息
字段 类型 描述
isBind boolean 是否绑定激活码）
code String 激活码
dealerId String 绑定的激活码对应的渠道ID
channelId String 绑定的激活码对应的服务商ID
ServiceConfigBean 字段信息
字段 类型 描述
service_url String 服务套餐地址
help_url String 帮助页面连接地址

获取用户的服务信息

接口说明
void getServiceInfoWithHandler(ITuyaResultCallback<DealerBean> callback)
参数说明

无

示例代码
TuyaSecurityServiceSDK.newInstance(homeId).getServiceInfoWithHandler(new ITuyaResultCallback<DealerBean>() {
        @Override
        public void onSuccess(DealerBean result) {
            // do something
        }

        @Override
        public void onError(String errorCode, String errorMessage) {
            // do something
        }
    });
